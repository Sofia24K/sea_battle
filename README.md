**Розробка гри "Морський бій"**  
  **Функціональні можливості:** 
- Розміщення кораблів на ігровому полі
- Перевірка коректності розташування кораблів
- Обробка ходів гравців
- Визначення влучання/промаху
- Автоматичне визначення переможця
- Консольний режим гри  
**Реалізована логіка:**
- Двовимірне ігрове поле(масив 10х10)
- Алгоритм перевірки влучань
- контроль завершення гри
- Обмеження повторних ходів по одній клітинці
- Перевірка правильності розміщення кораблів  
**Запуск проєкту:**
1. Відкрити проєкт у Visual Studio  
2. Обрати С++ проєкт
3. Запустити програму
